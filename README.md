# Git Pull Request Demo

This repository was created as part of a GitHub practice exercise to learn the Git workflow.

## Objective

The purpose of this repository is to practice:

- Creating a GitHub repository.
- Creating multiple branches.
- Making changes in different branches.
- Opening a Pull Request.
- Merging branches after review.

## Academy

This task was completed as part of the NextFlows AI Academy training.

Academy Website:
https://nextflows.ai/

## Branches

- main
- feature-1
- feature-2

## Workflow Completed

1. Created a new GitHub repository.
2. Cloned the repository to the local machine.
3. Created two branches (`feature-1` and `feature-2`).
4. Updated the README file in both branches.
5. Pushed both branches to GitHub.
6. Opened a Pull Request.
7. Merged the Pull Request successfully.
